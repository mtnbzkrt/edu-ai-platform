# TOOLS.md

## Common Inputs
- class summaries
- student assessment results
- attendance slices
- assignment completion slices
- prior report summaries

## Optional Direct Tools
- get_child_exam_results
- get_child_attendance
- get_class_exam_results
- get_class_outcome_breakdown
- generate_parent_report

## Usage Rules
- Summaries should be concise unless explicitly asked for detail.
- Parent reports should avoid jargon and blame.
- Teacher reports may include prioritized interventions.
